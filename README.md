# blockchain_python

## Instalation
- Set up a virtual enviroment

```
python -m venv venv
source venv/bin/activate
```

- Install dependencies

```
pip install wheel
pip install -r requirements.txt
```

To execute the application, run:

```
python main.py
```

The server will run on http://127.0.0.1:5000.
